🔌 DOCTOR ENDPOINTS REFERENCE
═══════════════════════════════════════════════════════════════════════════════

GET REQUESTS (All read endpoints):

GET /api/doctors
├─ Purpose: Get all doctors
├─ Auth: Bearer token required
├─ Params: None
└─ Response: Array of doctor objects

GET /api/doctors/1
├─ Purpose: Get specific doctor by ID
├─ Auth: Bearer token required
├─ Params: Replace 1 with doctor ID
└─ Response: Single doctor object

GET /api/doctors/user/2
├─ Purpose: Get doctor profile for a user
├─ Auth: Bearer token required
├─ Params: Replace 2 with user ID
└─ Response: Single doctor object

GET /api/doctors/license/DEN-2024-001
├─ Purpose: Search doctor by license number
├─ Auth: Bearer token required
├─ Params: Replace DEN-2024-001 with license number
└─ Response: Single doctor object

GET /api/doctors/specialization/Orthodontist
├─ Purpose: Get doctors filtered by specialty
├─ Auth: Bearer token required
├─ Params: Replace Orthodontist with specialization
└─ Response: Array of matching doctor objects

GET /api/doctors/available
├─ Purpose: Get only available doctors
├─ Auth: Bearer token required
├─ Params: None
└─ Response: Array of available doctor objects


POST REQUEST (Create):

POST /api/doctors
├─ Purpose: Create new doctor profile
├─ Auth: Bearer token required (Admin only)
├─ Body: JSON with doctor details
│  ├─ userId (required): Valid Doctor user ID
│  ├─ specialization (required): e.g., "Orthodontist"
│  ├─ licenseNumber (required, unique): e.g., "DEN-2024-001"
│  ├─ yearsOfExperience (required): e.g., 8
│  ├─ consultationFee (required): e.g., 500.00
│  ├─ bio (optional): Professional biography
│  ├─ department (optional): e.g., "Orthodontics"
│  ├─ clinicLocation (optional): e.g., "Main Branch"
│  └─ isAvailable (required): true/false
└─ Response: 201 Created with doctor object


PUT REQUEST (Update):

PUT /api/doctors/1
├─ Purpose: Update doctor details
├─ Auth: Bearer token required (Admin or Doctor own profile)
├─ URL Param: Replace 1 with doctor ID
├─ Body: JSON with fields to update (all optional)
└─ Response: 200 OK with updated doctor object


DELETE REQUEST (Delete):

DELETE /api/doctors/1
├─ Purpose: Delete doctor profile
├─ Auth: Bearer token required (Admin only)
├─ URL Param: Replace 1 with doctor ID
└─ Response: 204 No Content
